# 🌤️ Simple Weather App

A clean and responsive weather application built with HTML, CSS, and JavaScript.  
It fetches real-time weather data using the **OpenWeather API**.

![Weather App Screenshot](screenshot.png)

## 🔧 Features
- Live weather data for any city
- Responsive design (mobile & desktop)
- Weather icons and descriptions
- Error handling for invalid input
- Clean, modern UI

## 📦 Built With
- HTML5
- CSS3 (Flexbox & media queries)
- Vanilla JavaScript (Fetch API)
- [OpenWeather API](https://openweathermap.org/api)

3. Type a city and press “Get Weather”
